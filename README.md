# EcoPulse AI — Carbon Habit Coach

EcoPulse AI is a smart carbon footprint and eco habit tracker that helps users record sustainability actions, estimate carbon impact, unlock green badges, monitor weekly eco progress, and receive AI-style sustainability recommendations.

This project is fully static and can be deployed directly using GitHub Pages.

## Live Demo

Live project: https://danishzulkafli.github.io/ecopulse-ai-carbon-habit-coach/

## Estimated CO₂ Factors

The app uses simple educational estimates for carbon calculation:

- Transport: kg CO₂ per km
- Energy: kg CO₂ per kWh
- Food: kg CO₂ per meal
- Waste: kg CO₂ per kg
- Water: kg CO₂ per liter
- Shopping: kg CO₂ per item
- Digital Usage: kg CO₂ per hour
- Tree Planting: estimated yearly CO₂ absorption
